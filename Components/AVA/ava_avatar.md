# `ava.avatar` - Base Definition For VR & V-Tubing Avatars
Base component for VR & V-Tubing avatars.

## Json
| Key | Type | Description | Required | Default |
| --- | --- | --- | --- | --- |
| `auto` | boolean | Auto-map avatar features. | N | `true` |

**Example**
```
{
	"t": "ava.avatar",
	"auto": true
}
```
